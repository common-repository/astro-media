== Description ==
Horoscopen en Kaartleggingen is een GRATIS Nederlandstalige plugin van Astro Media welke je heel eenvoudig toevoegt aan je website. Over iedere verkoop welke via jouw website gerealiseerd wordt ontvang jij 25% commissie welke je kunt inzien in je persoonlijke dashboard.

Door deze Horoscopen en Kaartleggingen plugin toe te voegen ontvang jij:
-	Daghoroscoop
-	Maandhoroscoop
-	Relatiehoroscoop
-	Persoonlijke uitgebreide geboortehoroscoop
-	Toegang tot je persoonlijke dashboard
-	Binnenkort nog meer!

Door de plugin te installeren ontvang je een account in het Astro Media dashboard. Hier monitor je al je verkopen, en zie je direct het bedrag wat je aan commissie uitgekeerd krijgt.  
Astro Media is al 25 jaar marktleider op het gebied van Horoscopen en Kaartleggingen en partner van vele vakbladen, tijdschriften en websites, zowel online als offline.

Daghoroscoop
Kies je sterrenbeeld en ontdek wat de dag je zal brengen. Misschien wel veel liefde of geluk? Wanneer je iedere dag je horoscoop leest voordat je de dag gaat beginnen weet je wat jou te wachten staat.

Uitgebreide geboortehoroscoop
Vul je naam, geboortedatum, geboortetijdstip, geboorteland en geboorteplaats in. Op basis hiervan ontvang je een zeer uitgebreide astrologische karakterbeschrijving. Er wordt gekeken naar een groot aantal planeetstanden en de invloed die deze uitoefenen. De horoscoop bevat geen moeilijke astrologische begrippen, maar is geschreven in begrijpelijk Nederlands. 

Relatiehoroscoop
De relatiehoroscoop verkent het gebied van wat je samen deelt. Bestaat er een natuurlijke harmonie, of spelen er nog onuitgesproken zaken onder de oppervlakte? Hoor je alles wat tussen jullie gezegd wordt, of is er een boodschap tussen de regels? Meer dan alleen Mannen van Mars en Vrouwen van Venus onderzoekt de relatiehoroscoop ook de wisselwerking tussen jou en je liefdespartner of vriendschap.

Maandhoroscoop
Hier vind je de maandhoroscopen voor alle sterrenbeelden. Klik op jouw sterrenbeeld en lees wat deze maand je gaat brengen. Misschien wel veel geluk of misschien wel liefde? Als je aan het begin van iedere maand je maandhoroscoop leest weet je een beetje wat jou te wachten staat en kun je je hierop voorbereiden.


== Installation ==

1. Install the plugin as usual
2. Activate the plugin
3. Create an account via the button provided in the main menu
4. Use your credentials in the plugin
5. Select the shortcode you would like to use
6. Receive 25% commission per sale!
